# ⚙️ EcoWatch AI — Backend

## 📋 Responsabilidades do Backend

1. **Integração com APIs Públicas/Governamentais**:
   - Consumir dados do INMET, CPTEC/INPE, MapBiomas e OpenWeather.
2. **Integração com Gemma AI**:
   - Processar prompts de usuários e gerar relatórios ambientais estruturados.
3. **Servir Endpoints REST para o Frontend**:
   - Indicadores ambientais por município.
   - Alertas ativos e configuração de limiares.
   - Interface de comunicação com a IA Gemma.

Para detalhes completos da estrutura frontend e interfaces de dados, veja o arquivo [LEIAME.md no repositório principal](../LEIAME.md).
